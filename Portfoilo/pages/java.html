Student Examination Management System

Main Menu
Displays all available operations such as adding students, displaying records, searching, sorting, and exiting the system using a menu-driven console interface.
Add Student (Array + File Handling)
Allows the user to add new student details including name, roll number, and subject marks. The data is stored in arrays and also saved permanently using file handling.
Display Student Records
Displays complete student information including subject marks, total marks, and grade, which are automatically calculated using Java logic.
Search by Name (Linear Search)
Searches student records by name using the Linear Search algorithm and displays the matching student details.
Sort by Total Marks (Merge Sort)
Sorts all students in descending order based on total marks using the Merge Sort algorithm, helping to identify top performers easily.
Search by Roll Number (Binary Search)
Efficiently searches for a student using roll number with Binary Search after sorting, providing fast and accurate results.
